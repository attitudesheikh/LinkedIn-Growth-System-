## Key Rules

- **Never pitch in DM #1.** Build rapport first — pitch second.
- **Batch content on Sundays.** One session = an entire week.
- **Run hooks before every post.** The first line determines 80% of reach.
- **Fix the leakiest funnel stage first.** Don't optimize top of funnel when the bottom is broken.
- **Track only what you'll act on.** If a metric doesn't change what you do next week, stop tracking it.
- **Follow up 5–12 times.** 80% of sales happen between touchpoint 5 and 12.
